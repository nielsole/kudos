Kudos - Simple Self-Hostable Like button

Quickstart

1. Load the JS and CSS files

2. Embed them in your website


<script src="kudosplease.min.js"></script>
<link rel="stylesheet" href="kudosplease.css">

3. Create the like button at the position you like


<div class="kudos" data-amount="0" data-url="example.com/my-awesome-article"></div>

4. Create the Javascript object

<script>
new KudosPlease({
    el : '.kudos',
    duration : 1500,
    persistent : true,
    status : {
    alpha: 'fontelico-emo-shoot',
    beta: 'fontelico-emo-shoot',
    gamma: 'fontelico-emo-beer'
    }
});
</script>

5. If you want to self-host the backend, check out the instructions in the README
